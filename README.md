# 🚨 Sahayak-AI  
**Decentralized Real-Time Emergency Response Platform**

> Built for **Google Developer Group (GDG) Solution Challenge 2026** 🌍

Sahayak-AI is an AI-powered SOS broadcasting system that enables users to send emergency alerts in real time and receive assistance from nearby individuals. The platform prioritizes **local response (within 5 km)** while also allowing **global visibility of alerts**, creating a scalable and community-driven emergency network.

---

## 🌍 Problem Statement

In critical situations such as accidents, medical emergencies, or disasters, immediate help is often unavailable due to:

- Delayed response from centralized emergency services  
- Lack of awareness among nearby individuals  
- No real-time system connecting people in distress with responders  

---

## 💡 Solution

Sahayak-AI bridges this gap by:

- Broadcasting SOS alerts instantly  
- Notifying users within a **5 km radius for faster response**  
- Allowing users to **view all alerts globally using a toggle**  
- Using AI to understand and classify emergency situations  

This ensures both **rapid local assistance** and **wider community awareness**.

## ⚙️ Tech Stack

- **Frontend:** Antigravity, HTML, CSS, JavaScript  
- **Backend:** Firebase (Realtime Database / Firestore)  
- **Hosting:** Firebase Hosting  
- **AI:** Google Gemini API  
- **Maps & Location:** Google Maps API  
- **Authentication:** Google OAuth (Client ID)  
- **Cloud:** Google Cloud Platform  

---

## 🧠 How It Works

1. User clicks **Broadcast SOS**  
2. Enters emergency details  
3. Location is automatically detected  
4. Data is sent to Firebase  
5. Gemini AI processes and classifies the emergency  
6. SOS is broadcast to users  

**Filtering Logic:**
- Toggle ON → Alerts within 5 km  
- Toggle OFF → All global alerts  

7. Nearby users receive alerts  
8. Users respond and assist  

---

## 🏗️ Architecture Overview

User → Frontend → Firebase → Gemini API → Location Filtering → Broadcast → Responders
